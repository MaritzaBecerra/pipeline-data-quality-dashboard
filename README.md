# Pipeline Data Quality Dashboard

## Overview

This project is a fictional GIS portfolio project demonstrating how ArcGIS Pro, Power BI, Python, and Excel can be used together to manage pipeline assets and analyze data quality.

The project simulates a midstream pipeline company named **Lone Star Midstream**.

---

## Objectives

- Build a realistic pipeline geodatabase
- Perform GIS data quality checks
- Create executive dashboards in Power BI
- Automate GIS workflows with Python
- Document the entire workflow

- Power BI
- Python
- VS Code
